## 📋 Sobre o Projeto

Portfólio web interativo desenvolvido com **React + Vite + Tailwind CSS**, apresentando minha trajetória profissional, projetos e habilidades técnicas. Funciona como CV online com download em PDF e formulário de contato integrado.

🔗 **Deploy:** [gusjmo.vercel.app](https://gusjmo.vercel.app)

---

## ✨ Funcionalidades

- 🎨 **Design dark** com gradientes roxo/pink e animações suaves
- 📱 **100% responsivo** — mobile, tablet e desktop
- 💼 **Seção de Projetos** com filtro por categoria (Todos / Jogos / Web / QA)
- 📄 **CV Online** embutido + botão de download em PDF
- 📬 **Formulário de contato** funcional
- ⚡ **Deploy automático** via Vercel CI/CD (push na main = deploy)
- 🔍 **SEO otimizado** com meta tags e Open Graph

## 🚀 Rodando Localmente

```bash
# Clone o repositório
git clone https://github.com/gusjmo/portfolio-vercel.git

# Acesse a pasta
cd portfolio-vercel

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Acesse: `http://localhost:5173`

---

## 📦 Deploy

```bash
# Build de produção
npm run build

# Preview local do build
npm run preview
```

O deploy na Vercel é **automático** — qualquer push na branch `main` aciona o pipeline.
